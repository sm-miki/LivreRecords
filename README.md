# LivreRecords

## 概要

書籍などを購入した際のレシート画像を読み取り、何を購入したかを管理するアプリ。Webアプリとして動作。

## 開発環境

- Python3
- Django
- Tesseract(未定)

## 機能

* レシート情報を元に購入記録を登録
* レシート情報を画像から読み取り可能
* 購入記録や書籍の情報を一覧で表示
